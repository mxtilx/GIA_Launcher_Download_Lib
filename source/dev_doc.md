# 命名格式

## 文件命名方式（暂定）

| 名称           | 意义                       |
|--------------|--------------------------|
| `listing`    | 顶级键鼠事件监听线程               |
| `static`     | 静态全局功能线程，只存在一个           |
| `flow`       | 流程控制线程，下属线程为`controller` |
| `controller` | 功能管理线程，下属线程为`operator`   |
| `operator`   | 具体功能执行线程                 |
